# **InvestAnalytics**

## Preqreqs for contribution
- Node.js (v18 or higher)
- npm or yarn
- Supabase

## Stack:
 - **Front End**: React tsx
 - **Auth**: Supabase auth
 - **Database**: Supabase postgres db
 - **APIs**: Serverless functions + external
 - **Backend**: Flask
